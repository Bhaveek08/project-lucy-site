# L.U.C.Y - Portfolio

**Lucent Unbound Conscious Yield**

Welcome to the official portfolio website of L.U.C.Y. This project showcases a custom-built, modern, premium aesthetic using pure Vanilla web technologies.

## Overview

This website is designed with a sleek Red, White, and Black theme. It relies on advanced CSS features and Vanilla JavaScript to create interactive, hardware-accelerated animations—completely avoiding the need for heavy frameworks or build tools.

## Key Features

- **Vanilla HTML/CSS/JS**: No React, no build steps, no NPM packages. Just pure, performant web standards.
- **Scroll Collapse Animation**: A dynamic scroll event seamlessly collapses "Lucent Unbound Conscious Yield" into the initials "L.U.C.Y".
- **Custom Particle Effect**: A lightweight `<canvas>` based background effect that adds depth and movement.
- **Glassmorphism Design**: Frosted glass panels and layered UI elements for a premium look.
- **Responsive Fluid Typography**: Automatically adjusts for any screen size, ensuring a perfect layout on both PC and mobile devices.
